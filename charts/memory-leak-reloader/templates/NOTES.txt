{{- else -}}
Enforcing: workloads with a MemoryLeakPolicy will be rollout-restarted when leaking
(subject to cooldown, maintenance windows, and the global concurrency cap).
{{- end }}

Opt a workload in by creating a MemoryLeakPolicy that references it:
  apiVersion: memreload.io/v1alpha1
  kind: MemoryLeakPolicy
  metadata:
    name: api
    namespace: <ns>
  spec:
    workloadRef:
      kind: Deployment
      name: api

Inspect state with:  kubectl -n <ns> get mlp
